WYSI? What's that?

In this context, WYSI, or "when you see it," refers to the ongoing joke within the osu! game community. This expression is often paired with the number 727, referring to the number of performance points (pp) awarded to scores set by players Cookiezi and Aireu in multiple occasions.

See this wiki page to learn more.

Why this domain?

I bought this domain and asked the community what I should do with it. The top comment suggested that it redirect to Aireu's Twitch channel, so here's that obligatory link.

For a while, a direct redirect to that channel was all this domain did. That is, until I started searching for a good short link domain and realized that this was a perfect candidate. At that point, I made the choice to use this domain for short links instead.
